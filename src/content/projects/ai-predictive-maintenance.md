## Overview

Built a machine learning-powered predictive maintenance system capable of processing multivariate industrial sensor streams for anomaly detection and Remaining Useful Life estimation.

## Approach

Implemented Isolation Forest and regression-based models to analyse machine conditions and estimate operational degradation.

Engineered drift-resistant features to improve anomaly detection robustness across changing industrial environments.

Integrated a FastAPI backend with a Streamlit dashboard for real-time monitoring, prediction serving, and machine health visualization.

## Highlights

- Real-time monitoring dashboard
- Backend-integrated ML pipeline
- Remaining Useful Life estimation
- Drift-resistant anomaly detection
