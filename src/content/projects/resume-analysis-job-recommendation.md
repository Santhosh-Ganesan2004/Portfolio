## Overview

Designed an NLP-powered resume analysis system using OCR preprocessing and semantic embedding workflows.

## Approach

Implemented modular extraction pipelines for processing resume documents and generating structured candidate insights.

Used SBERT embeddings with cosine similarity scoring to compute candidate-role matching, achieving 83% validation accuracy.

Built an interactive Streamlit dashboard for profile scoring, recommendation visualization, and resume analytics.

- 83% validation accuracy

- OCR preprocessing pipeline
