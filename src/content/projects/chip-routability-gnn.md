## Overview

Developed a Graph Neural Network-based pipeline for analysing IC layout routability using graph representations and congestion-aware classification techniques.

## Approach

Converted more than 170 IC layouts into structured graph-based datasets and trained a GCN model for routability prediction.

Generated pseudo-labels using congestion and DRC threshold analysis to enable supervised learning on partially unlabeled datasets.

Built a custom inference workflow for producing routability heatmaps that visually separate congested and optimized layouts.

## Highlights

- Custom graph-based inference workflow
